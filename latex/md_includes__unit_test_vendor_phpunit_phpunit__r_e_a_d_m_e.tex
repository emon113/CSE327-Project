PHPUnit is a programmer-\/oriented testing framework for PHP. It is an instance of the x\+Unit architecture for unit testing frameworks.

{\texttt{ }} {\texttt{ }} {\texttt{ }} {\texttt{ }} {\texttt{ }}\doxysection{Installation}\label{md_includes__unit_test_vendor_phpunit_phpunit__r_e_a_d_m_e_autotoc_md298}
We distribute a {\texttt{ PHP Archive (PHAR)}} that has all required (as well as some optional) dependencies of PHPUnit 9.\+0 bundled in a single file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ wget https://phar.phpunit.de/phpunit-\/9.0.phar}
\DoxyCodeLine{}
\DoxyCodeLine{\$ php phpunit-\/9.0.phar -\/-\/version}

\end{DoxyCode}


Alternatively, you may use {\texttt{ Composer}} to download and install PHPUnit as well as its dependencies.
