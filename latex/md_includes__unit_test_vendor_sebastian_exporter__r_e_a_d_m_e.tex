This component provides the functionality to export PHP variables for visualization.\doxysection{Installation}\label{md_includes__unit_test_vendor_sebastian_exporter__r_e_a_d_m_e_autotoc_md448}
You can add this library as a local, per-\/project dependency to your project using {\texttt{ Composer}}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{composer require sebastian/exporter}

\end{DoxyCode}


If you only need this library during development, for instance to run your project\textquotesingle{}s test suite, then you should add it as a development-\/time dependency\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{composer require -\/-\/dev sebastian/exporter}

\end{DoxyCode}
