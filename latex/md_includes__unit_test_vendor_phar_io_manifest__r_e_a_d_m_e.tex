Component for reading {\texttt{ phar.\+io}} manifest information from a {\texttt{ PHP Archive (PHAR)}}.

{\texttt{ }} {\texttt{ }} {\texttt{ }}\doxysection{Installation}\label{md_includes__unit_test_vendor_phar_io_manifest__r_e_a_d_m_e_autotoc_md34}
You can add this library as a local, per-\/project dependency to your project using {\texttt{ Composer}}\+: \begin{DoxyVerb}composer require phar-io/manifest
\end{DoxyVerb}
 If you only need this library during development, for instance to run your project\textquotesingle{}s test suite, then you should add it as a development-\/time dependency\+: \begin{DoxyVerb}composer require --dev phar-io/manifest
\end{DoxyVerb}
